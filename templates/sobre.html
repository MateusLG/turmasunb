Sobre o TurmasUnB

Como surgiu, como funciona e quem fez

Como surgiu

O TurmasUnB nasceu de uma necessidade real: o gruposfga, projeto criado pelo Guilherme Puida para centralizar os links de grupos de turmas da Faculdade do Gama (FGA), saiu do ar quando o Guilherme se formou.

A ideia era boa demais para morrer. Então expandi o conceito para cobrir toda a UnB — todas as 210 unidades, mais de 6.400 turmas — e abri para qualquer aluno contribuir com o link do grupo da sua turma.

Como funciona

Os dados das turmas são extraídos diretamente do SIGAA a cada semestre usando um scraper automático. Cada turma começa sem link — qualquer pessoa pode entrar no site e adicionar o link do grupo (WhatsApp, Telegram, Discord, Meet ou qualquer outro).

Não tem cadastro, não tem login. A ideia é ser simples e rápido: buscou a matéria, encontrou a turma, entrou no grupo.

Como foi feito

Projeto open source, feito com tecnologias simples e sem fricção:

Python 3.12 linguagem principal do backend e do scraper
FastAPI framework web que serve as rotas e a API JSON
Jinja2 renderização dos templates HTML no servidor
Tailwind CSS estilização do frontend via CDN, sem build step
Playwright scraper headless que coleta as turmas do SIGAA
PostgreSQL banco de dados que armazena os links cadastrados
Railway hospedagem do servidor e do banco de dados
Cloudflare DNS e domínio turmasunb.com
